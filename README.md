# ESLINTRC
ESLint configuration

## Install
Yarn:
```bash
yarn add -D \
    @ravorona/eslint-config \
    eslint \
    eslint-plugin-const-case
```
NPM:
```bash
npm i --save-dev \
    @ravorona/eslint-config \
    eslint \
    eslint-plugin-const-case
```

## Usage
Update extends property inside your stylelint configuration file

```json
{
    "extends": "@ravorona"
}
```
